# Swapi Application

## Descriptions
Application using starwars API https://swapi.co/ to accessible data source.

## Getting Started
The app using react, reactstrap, redux, redux-logger, redux-thunk, and react-router-dom.

## How to use
```
> git clone https://github.com/danilags/swapi.git
> npm install
> npm start
```
